## Output - 输出
- 在屏幕上打印一串文字
在pseudocode里面伪代码如何使用output？
什么时候需要用到output？ 就是当你需要把结果或者其中的过程展示给别人看的时候
也可以output用来输出提示词

``` VB
OUTPUT "Hello,world!"

OUTPUT "abcdefg"

OUTPUT "1234"

DECLARE code: STRING
OUTPUT "请输入你的验证码:"
INPUT code      // 这样就表示输入的验证码会放到yanzhengma这个变量里面去
```

## INPUT  - 输入
- 代码在执行到input语句的时候会等待用户输入，当用户输入完之后会把用户输入的内容加载到对应**变量(Variable)**里面


## 变量 - 可变的量 - 用来存储一个值，并且随时可以被更改
- 用一个符号，也就是变量名来代表这个值
